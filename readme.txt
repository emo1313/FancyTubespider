SME Tube Spider  -  allow your visitors to search videos in youtube , Integrates search & play , gives preview of videos on mouseover, and more.
INSTALLATION:
------------------
unzip all files in a folder!

open file config.php change the variables

// URL RUNNING THE SCRIPT	
$_URL="http://labs.mydomain/tube_spider";
// DEFAULT SEARCH DISPLAY VIDEOS BASED IN THIS KEYWORD IN INDEX
$_dv="Funny";


send all files from your server
